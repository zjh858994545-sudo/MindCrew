# MindCrew 验收提问清单

## 使用说明

本清单用于在知识文档导入完成后，对问答效果做分组验收。建议做法：

1. 先不指定知识库，测试全局召回
2. 再指定单个知识库，测试范围过滤
3. 对回答同时检查：
   - 是否答到点上
   - 是否给出来源
   - 来源是否与问题匹配
   - 在结果不足时是否明确说明

## 十、验收通过建议标准

可以参考以下简单标准：

- 70% 以上问题能直接答中核心要点
- 关键问题能给出正确来源
- 结果不足时能明确承认不足，而不是编造
- 指定知识库范围后，回答明显更聚焦
- 多文档综合问题能给出结构化答案
