# 企业文档：版本说明样例

## 文档目标

本文档提供一份接近企业实际使用的版本说明样例，用于帮助团队沉淀规范化的发布记录，也可作为知识库的版本类问答语料。

---

## 版本 `v1.2.0`

### 发布时间

2026-02-20

### 主要更新

- 新增知识库范围选择器
- 支持会话导出 Markdown
- 优化文档处理任务状态展示

### 修复项

- 修复部分 Word 文档切片顺序错误问题
- 修复 SSE 长连接偶发中断问题

### 升级说明

- 无数据库结构变更
- 无强制配置变更

### 已知限制

- 网页搜索功能默认关闭
- 大体量 PDF 文档处理耗时仍较高

---

## 版本 `v1.3.0`

### 发布时间

2026-03-05

### 主要更新

- 上线 Agentic RAG 主链路
- 新增长期记忆读写能力
- 新增 MCP 工具注册与外部暴露
- 支持知识库与网页来源统一输出

### 兼容性说明

- 原有问答接口保持可用
- 旧会话数据无需迁移即可读取
- 新增 MCP 能力不影响原有前端页面使用

### 配置变更

新增可选配置：

- 网页搜索开关
- 外部搜索地址
- 外部搜索工作空间
- 外部搜索 API Key

### 风险提示

- 若未配置外部搜索，`web_search` 会自动降级为空结果
- 若向量库为空，系统可能频繁走兜底回答

---

## 版本 `v1.4.0`

### 发布时间

2026-03-31

### 主要更新

- 提示词与文案统一调整为通用知识库语境
- 前端用户画像页面改为通用偏好与背景信息管理
- 默认资源命名从历史兼容命名切换为 `mindcrew`
- 增加企业规范、接口、部署、版本说明类示例文档

### 升级步骤

1. 检查配置项是否仍使用历史默认值
2. 确认对象存储 Bucket 名和向量集合名
3. 重新导入示例知识文档
4. 执行问答验收清单

### 建议验收项

- 登录与知识库上传
- 指定知识库问答
- 来源展示
- 网页搜索降级逻辑
- MCP 工具可见性

### 已知问题

- 历史测试数据和设计文档中仍保留部分旧示例内容
- 兼容层中仍存在少量历史类名和表结构命名

## 总结

规范的版本说明可以帮助团队快速理解“本次上线带来了什么、需要改什么、有哪些风险、出现问题如何回滚”。对于知识库问答系统，这类文档本身也非常适合作为版本相关问题的知识来源。
